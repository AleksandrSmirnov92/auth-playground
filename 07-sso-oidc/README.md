# 07 - SSO / OpenID Connect

**Быстрый старт (после реализации):**

- **Go:** `cd 07-sso-oidc/go && go run cmd/server/main.go`
- **Node.js:** `cd 07-sso-oidc/node && npm install && npm run dev`
- **C# .NET:** `cd 07-sso-oidc/csharp && dotnet run`

**Swagger:** `http://localhost:8080/swagger`

Single Sign-On через OpenID Connect. Discovery endpoint, ID Token, UserInfo.
