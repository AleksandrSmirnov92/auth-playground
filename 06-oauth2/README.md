# 06 - OAuth 2.0

**Быстрый старт (после реализации):**

- **Go:** `cd 06-oauth2/go && go run cmd/server/main.go`
- **Node.js:** `cd 06-oauth2/node && npm install && npm run dev`
- **C# .NET:** `cd 06-oauth2/csharp && dotnet run`

**Swagger:** `http://localhost:8080/swagger`

Авторизация через внешних провайдеров (Google, GitHub). Authorization Code Flow.
