# 03 - Session + Cookie

**Быстрый старт (после реализации):**

- **Go:** `cd 03-session-cookie/go && go run cmd/server/main.go`
- **Node.js:** `cd 03-session-cookie/node && npm install && npm run dev`
- **C# .NET:** `cd 03-session-cookie/csharp && dotnet run`

**Swagger:** `http://localhost:8080/swagger`

Серверные сессии с идентификатором в Cookie. Login создаёт сессию, Logout уничтожает.
